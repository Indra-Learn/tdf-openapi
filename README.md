
1. First clone the blank repo, created in github:
    ```bash
    git clone https://github.com/Indra-Learn/tdf-openapi.git
    cd tdf-openapi
    ```
2. After installing **node.js**, create `next.js` project setup by below command:
